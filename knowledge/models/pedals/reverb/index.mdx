---
title: Reverb
description: Reverb effect models and validation targets.
---

Reverb models should make time-domain memory explicit while preserving the chain boundary model: input loading, output source impedance, bypass behavior, and stable runtime control updates.

Current models:

- Springfield: spring-tank-inspired reverb with dwell, tone, mix, and FX-loop placement.
- StudioVerb: original digital room/plate reverb with FDN diffusion, predelay,
  tone, low-cut, modulation, and wet/dry mix.
