# Greybound Lab Models

This directory is for local experimental neural-cell artifacts generated by the
lab. Model descriptors, weights, ONNX exports, checkpoints, plots, and reports
are local research outputs unless they have been explicitly reviewed and
promoted.

Keep large artifacts out of git. Commit only small source-safe manifests,
schemas, experiment notes, and accepted documentation.

The planned accepted artifact shape is:

```text
model.greybound.json
weights.greybound.bin
```

The descriptor schema is:

```text
lab/schemas/neural-cell-artifact.schema.json
```

ONNX exports may be generated for inspection and tool compatibility, but they
are not the source of truth for the live Rust runtime.
